## Maintenance Tasks
- Review and merge pull requests
- Update dependencies and requirements.txt as needed
- Ensure new devices follow the per-device folder structure
- Keep documentation up to date (README, architecture, usage, contributing)
- Monitor for bug reports and address issues promptly

## Release Process
1. Ensure all tests pass and code is linted
2. Update version and changelog
3. Tag release in version control
4. Announce release to users/maintainers
